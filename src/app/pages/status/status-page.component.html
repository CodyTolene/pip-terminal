Welcome to the Pip-Boy Terminal!

This is a fan-made terminal interface designed to interact with your Pip-Boy 3000 Mk V.

You can contribute freely to this project if you have any ideas or suggestions by visiting GitHub. This project is open-source and welcomes contributions from everyone all across the wasteland!

If you have any questions, feel free to reach out to your nearest overseer, or the Discord channel.

Use this site to connect, add apps, games, browse files, update files, unlock hidden features of your device, and more! Need the official unit? You can purchase the Pip-Boy 3000 Mk V directly from the Bethesda Gear Store.

For official firmware updates, visit The Wand Company.
